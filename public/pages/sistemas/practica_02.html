Práctica de comandos:

 1- LISTAR EL DIRECTORIO RAÍZ 


Con el comando ls /  listamos todo lo que hay en la raiz del sistema


2- CREAR ESTRUCTURA SIGUIENTE DESDE EL DIRECTORIO /home/usuario. 


Con el comando mkdir se crean carpetas y con el comando tree visualizamos la estructura creada.


3- CREAR EN /home/usuario UN FICHERO F1 CON LOS DATOS PERSONALES DEL ALUMNO QUE HACE LA PRÁCTICA 


El comando pwd nos dice donde estamos ubicados, es decir, en que carpeta nos encontramos.


El comando nano es un editor de texto en terminal al ejecutarlo podemos abrir un archivo o crearlo de 0. Con el comando touch creamos archivos.




Con el comando cat, mostramos en pantalla el contenido de un archivo.



4- MOVER EL FICHERO F1 AL DIRECTORIO DE TRABAJO /home/usuario/practica


Usamos el  comando mv para mover ficheros desde una ubicación a otra.

Con tree mostramos la estructura de carpetas



5- COPIAR EL FICHERO F1 EN EL DIRECTORIO DIR11 


El comando cp sirve para copiar ya sea archivos o carpetas de una ubicación a otra.

El comando ls y la ubicación mostramos el contenido de la carpeta

6- CREAR EL DIRECTORIO COPIA DENTRO DEL DIRECTORIO USUARIO Y COPIAR EN ÉL EL CONTENIDO DE DIR1. 


Con cp -R copiamos la carpeta y el contenido de la misma ya que -R es recursivamente.

7- DESDE /home/usuario HAZ UN LINK (I-NODO) DEL FICHERO EN EL DIRECTORIO DIR11 Y DEMUESTRA SU EXISTENCIA.

ln -s es el comando que se utiliza para crear enlaces / accesos directos

8- CREA UNA CARPETA EN /home/usuario QUE SE LLAME ENLACES Y CONTENDRÁ ENLACES A CADA UNO DE LOS ELEMENTOS DE DIR1 


Con ln -s “/ubicación/origen/archivo”  “/ubicación/destino/archivo”  ubicamos directamente el enlace en el destino.



9- EN LA GESTIÓN DE LINUX ES MUY IMPORTANTE VER QUÉ PASA EN EL SISTEMA, IGUAL QUE EN WINDOWS CON EL REGISTRO DE SUCESOS. 

EN LINUX SE LLAMA /var/log/syslog Y SE TRATA DE UN ARCHIVO DONDE SE REGISTRA TODO LO QUE OCURRE EN EL SISTEMA. 

DESACTIVA UNA INTERFACE (P.EJ. ETH0 CON ip link set (INTERFACE p.e. enp0s3) down Y LUEGO VUELVE A ACTIVARLA ip link set (INTERFACE p.e. enp0s3) up ), MUESTRA LAS ÚLTIMAS LÍNEAS DEL ARCHIVO /var/log/syslog, QUE NOS INDICARÁ QUE HA HECHO EL SISTEMA


El comando ip link nos muestra las interfaces del sistema operativo. 

Para poder apagar la interfaz de red debemos ejecutar el comando ip link set “nombre interfaz” down. Si somos root podemos ejecutarlo directamente, pero si no debemos poner lo siguiente:

sudo ip link set “nombre interfaz” down -> asi ejecutamos el comando como superusuario.





 Con cat /var/log/syslog veremos en el terminal lo sucedido en el sistema

Conclusiones

Esta práctica nos ayuda a comprender que la terminal de comandos se puede realizar lo mismo que en la interfaz gráfica, aparte de poder realizar de manera más eficiente algunas tareas como crear carpetas o ficheros rápidamente o a la vez. También  es una manera de consumir menos recursos del sistema.
